# 研修動画ライブラリ 開発履歴

## プロジェクト概要

- **目的**：介護・医療施設の研修動画に正確な文字起こしテキスト＋やさしい日本語を提供
- **対象**：日本人スタッフ＋外国人スタッフ（Chrome翻訳で母国語化）
- **公開URL**：https://kouga-k.github.io/kenshu-videos/
- **管理画面**：https://kouga-k.github.io/kenshu-videos/admin.html
- **管理者パスワード**：kenshu2026
- **GitHub**：https://github.com/kouga-k/kenshu-videos

## 構成

```
GitHub Pages（kouga-k.github.io/kenshu-videos）
├ index.html   ← 視聴者向け一覧
├ video.html   ← 視聴ページ（動画＋テキスト）
├ admin.html   ← 管理画面
└ Firebase Firestore（kenshu-videos-app）

      ├ title
      ├ driveFileId（Googleドライブの動画ID）
      ├ status（draft / review / published）
      ├ textPlain（原文テキスト）
      ├ textEasy（やさしい日本語）
      └ createdAt
```

## エラー回避履歴

### 1. Firestoreパーミッションエラー
**症状**：`Missing or insufficient permissions`
**原因**：Firestoreセキュリティルールがデフォルトで全拒否
**対処**：`firestore.rules` を以下に設定
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /videos/{doc} {
      allow read: if true;
      allow write: if true;
    }
  }
}
```

### 2. Firestore複合インデックスエラー
**症状**：視聴ページが「Firebase設定が必要です」エラー
**原因**：`where('status','==','published') + orderBy('createdAt','desc')` の組み合わせには複合インデックスが必要
**対処**：`orderBy`を削除しJavaScript側で並び替え
```javascript
const q = query(collection(db,'videos'), where('status','==','published'));
const videos = [];
snapshot.forEach(doc => videos.push({ id: doc.id, ...doc.data() }));
videos.sort((a, b) => (b.createdAt?.seconds || 0) - (a.createdAt?.seconds || 0));
```

### 3. ローカルファイル（file://）でFirebase動作不可
**症状**：`Unsafe attempt to load URL... 'file:' URLs are treated as unique security origins`
**原因**：Firebase SDKは`file://`プロトコルでは動作しない
**対処**：必ずGitHub Pages URL（`https://kouga-k.github.io/...`）で開く

### 4. Gemini API 404エラー
**症状**：複数のモデル名で全て404
- `v1beta/models/gemini-2.0-flash` → 404
- `v1beta/models/gemini-1.5-flash-latest` → 404
**原因**：未特定（APIキーの権限制限と推定）
**対処**：Gemini自動生成は諦め、Claudeチャット手動 or Google AI Studio直接利用を検討中

### 5. 動画埋め込み字幕の修正不可
**症状**：動画ファイル自体に誤った字幕が焼き込まれている
**原因**：YouTube/ドライブの自動字幕が動画にエンコード済み
**対処**：テキスト欄に正しい原文・やさしい日本語を表示することで補完。動画字幕は無視。

### 6. 文ハイライト機能がChrome翻訳で全文黄色化
**症状**：Chrome翻訳をかけると、全文が黄色ハイライトになる
**原因**：Chrome翻訳がspan要素を破壊・再構築するため、IDが消える・charIndexが原文と一致しない
**対処**：spanベースのハイライトを廃止。代わりに音声再生のみ提供。

### 7. 声リストが「読み込み中...」のまま
**症状**：`voiceSelect`が初期表示のまま更新されない
**原因**：`onvoiceschanged`イベント発火時にDOM要素が未生成
**対処**：コンテンツ生成後に`setTimeout(loadVoices, 100)`で再呼び出し

### 8. 音声が再生されない
**症状**：読み上げボタンを押しても音が出ない
**原因（推定）**：`setTimeout`で`speak()`を遅延実行するとChromeがユーザー操作コンテキスト外と判定しブロック
**対処**：`setTimeout`を削除し同期的に`speak()`を呼び出し

## 重要な制約・注意事項

### セキュリティ
- **APIキーは絶対にチャットに貼り付けない**
- **有料APIは事前に許可を取らない限り使用禁止**

### ファイル更新方法
1. ローカルファイル（`C:\Users\User\Desktop\プロジェクト\kenshu-videos\`）を編集
2. `git add` → `git commit` → `git push origin main`
3. 1〜2分でGitHub Pagesに反映

### Firebase設定
- プロジェクトID：`kenshu-videos-app`
- 設定情報は各HTMLファイル内に直書き
- Firestoreは無料枠（Spark）で運用

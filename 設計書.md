# 研修動画字幕・やさしい日本語アプリ 設計書 v4

最終更新：2026-05-27

---

## 1. 目的

介護・医療施設の研修動画に、正確な文字起こしテキストと「やさしい日本語」版を提供する。
外国人スタッフがChrome翻訳で母国語に変換して研修内容を理解できるようにする。
専門用語はそのまま残し、やさしい日本語で意味を補足して学習を支援する。

---

## 2. ユーザー

| 種類 | 役割 |
|------|------|
| 管理者 | 動画追加・文字起こし・訂正・やさしい日本語生成・公開 |
| 一般スタッフ | 動画視聴・テキスト読む・Chrome翻訳で母国語変換・音声で聞く |

## 4. テキストの3層構造

```
① 原文テキスト（専門用語そのまま）
   多職種連携を推進する中、看護師の役割拡大が求められており

② やさしい日本語版
   いろいろな職種の人が一緒に働く中で、看護師がする仕事が増えています

③ Chrome翻訳（ユーザーが任意でON）
   → ミャンマー語・ベトナム語・インドネシア語など

④ 音声読み上げ（無料・OS標準音声）
   → 各層を音声で聞ける。翻訳後の言語にも対応
```

---

## 5. やさしい日本語のルール

Gemini／Claudeへの指示に含める内容：

- 専門用語はそのまま残す（覚えてもらうため）
- 難しい表現をひらがな・やさしい言葉に言い換える
- 一文を短くする
- 主語をはっきりさせる
- 受け身表現を減らす

例：
| 原文 | やさしい日本語 |
|------|--------------|
| 多職種連携を推進する | いろいろな職種の人が協力する |
| 療養上の世話を行います | 患者さんの日常生活を助けます |
| 看護補助業務を実施する責任があります | 看護補助の仕事を安全にする責任があります |

## 9. 音声読み上げ機能

### 仕組み
- **Web Speech API**（ブラウザ標準・無料）を利用
- `speechSynthesis.speak()` でテキストを音声化
- システムにインストールされた音声を使用

### 対応音声例（Windows 11）
| 言語 | 音声名 | 品質 |
|------|--------|------|
| 日本語 | Microsoft Nanami / Kenji | ⭐⭐⭐⭐⭐ 自然 |
| 日本語 | Microsoft Ayumi / Haruka | ⭐⭐⭐ 標準 |
| 英語 | Microsoft Mark / Zira | ⭐⭐⭐⭐ 自然 |
| インドネシア語 | Microsoft Andika | ⭐⭐⭐⭐ 自然 |
| ベトナム語 | Microsoft An | ⭐⭐⭐⭐ 自然 |
| その他 | Google系 ☁ | ⭐⭐⭐⭐ クラウド経由 |

### 設計判断
- **文ハイライトは廃止**：Chrome翻訳でDOM構造が破壊されるため
- **声リストは現在のページ言語を最上位に表示**：Chrome翻訳後の言語に自動追従

## 10. 用語辞書（誤認識補正）

文字起こし時の誤変換補正リスト。Geminiプロンプトに組み込む予定。

| 誤認識 | 正しい表記 |

| 景観栄養 | 経管栄養 |
| 義脂 | 義歯 |
| 事女具 | 自助具 |
| 異常（移乗の文脈で）| 移乗 |
| 原点 | 減点 |
| 正式 | 清拭 |
| 上位・回（衣服の文脈で）| 上衣・下衣 |
| 遭具 | 装具 |

## 14. 重要ルール

1. **APIキーをチャットに貼り付けない**
2. **有料APIは事前許可なく使用しない**
3. **動画字幕（焼き込み済み）は修正不可** → テキスト欄で正しい内容を表示
4. **GitHub Pagesは1〜2分の反映遅延あり** → 強制再読み込み（Ctrl+Shift+R）で確認
5. **必ずHTTPS URL（kouga-k.github.io）でアクセス** → ローカル file:// では動かない

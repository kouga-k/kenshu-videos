# 研修動画字幕・やさしい日本語アプリ 設計書 v3

## 更新理由
- 全文ルビを削除（Chrome翻訳との相性問題・読みにくさ）
- やさしい日本語版テキストを追加（翻訳精度・理解度向上）
- アプリの責任範囲をさらに絞り込み

---

## 1. 目的

介護・医療施設の研修動画に、正確な文字起こしテキストと「やさしい日本語」版を提供する。
外国人スタッフがChrome翻訳で母国語に変換して研修内容を理解できるようにする。
専門用語はそのまま残し、やさしい日本語で意味を補足して学習を支援する。

---

## 2. ユーザー

| 種類 | 役割 |
|------|------|
| 管理者 | 動画追加・文字起こし・訂正・やさしい日本語生成・公開 |
| 一般スタッフ | 動画視聴・テキスト読む・Chrome翻訳で母国語変換 |

---

## 3. 機能一覧

### 必須機能
- 動画表示（GoogleドライブURL埋め込み）
- 文字起こし自動生成（Gemini API）
- 文字起こし内容の確認・訂正（管理者）
- やさしい日本語版テキスト自動生成（Gemini API）
- やさしい日本語版の確認・訂正（管理者）
- 動画一覧ページ
- 管理者パスワード保護
- 確認フラグ（下書き・確認中・公開）

### やらないこと
- 翻訳API連携（Chrome翻訳に任せる）
- 全文ルビ
- SRT字幕生成
- 多言語テキスト保存
- 言語切り替えボタン

---

## 4. テキストの3層構造

```
① 原文テキスト（専門用語そのまま）
   多職種連携を推進する中、看護師の役割拡大が求められており

② やさしい日本語版
   いろいろな職種の人が一緒に働く中で、看護師がする仕事が増えています

③ Chrome翻訳（ユーザーが任意でON）
   → ミャンマー語・ベトナム語・インドネシア語など
```

スタッフは①②を見ながらChrome翻訳で母国語に変換できる。

---

## 5. やさしい日本語のルール

Geminiへの指示に含める内容：

- 専門用語はそのまま残す（覚えてもらうため）
- 難しい表現をひらがな・やさしい言葉に言い換える
- 一文を短くする
- 主語をはっきりさせる
- 受け身表現を減らす

例：
| 原文 | やさしい日本語 |
|------|--------------|
| 多職種連携を推進する | いろいろな職種の人が協力する |
| 療養上の世話を行います | 患者さんの日常生活を助けます |
| 看護補助業務を実施する責任があります | 看護補助の仕事を安全にする責任があります |

## 9. 用語辞書（誤認識補正）

文字起こし時の誤変換を自動補正するリスト（Geminiプロンプトに組み込む）

| 誤認識 | 正しい表記 |
|--------|-----------|
| 点灯 | 転倒 |
| 離章 | 離床 |
| 食会 | 食介 |
| じょくそう | 褥瘡 |
| 店頭 | 転倒 |
